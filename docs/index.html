Getting Started

Lightgallery是一个轻量级的模块化响应灯箱长廊，它允许您创建美丽的形象和视频画廊。

Browser support

lightgallery支持所有主要的浏览器IE9，包括以上.

安装

安装 Bower

您可以安装lightgallery使用 Bower包管理器.

$ bower install lightgallery.js

安装npm

你也可以找到lightgallery上npm.

$ npm install lightgallery.js

从GitHub下载

您也可以直接从下载lightgallery github.

包括CSS和JavaScript文件

首先，在加lightgallery.css <head> 该文件

<head>
    <link type="text/css" rel="stylesheet" href="css/lightgallery.css" /> 
</head>

然后包括lightgallery.min.js到您的文档.
如果你想包括任何lightgallery插件后可以将其包含 lightgallery.min.js.

<body>
    ....

    <script src="js/lightgallery.min.js"></script>

    <!-- lightgallery plugins -->
    <script src="js/lg-thumbnail.min.js"></script>
    <script src="js/lg-fullscreen.min.js"></script>
</body>

Lightgallery还支持AMD，CommonJS的和ES6模块

当您使用AMD确保lightgallery.js之前lightgallery模块加载.

require(['./lightgallery.js'], function() {
    require(["./lg-zoom.js", "./lg-thumbnail.js"], function(){
        lightGallery(document.getElementById('lightgallery')); 
    });
  });

The markup

lightgallery不强迫你使用任何类型的标记。你可以使用任何你想要的标记。但我建议你使用以下的标记. Here 你可以找到不同类型的加价的详细的例子.

<div id="lightgallery">
  <a href="img/img1.jpg">
      <img src="img/thumb1.jpg" />
  </a>
  <a href="img/img2.jpg">
      <img src="img/thumb2.jpg" />
  </a>
  ...
</div>

调用插件

最后，你需要通过添加下面的代码以启动图库.

<script type="text/javascript">
    lightGallery(document.getElementById('lightgallery')); 
</script>

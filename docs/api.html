事件

如何使用lightgallery自定义事件 

var lg = document.getElementById('lightgallery');

// Perform any action just before opening the gallery
lg.addEventListener('lgOnBeforeOpen', function(e){
  alert('onBeforeOpen');
}, false);

// custom event with extra parameters 
// event.detail.index -滑动指数
// event.detail.fromTouch - true if slide function called via touch event or mouse drag
// event.detail.fromThumb - true if slide function called via thumbnail click
lg.addEventListener('lgOnBeforeSlide', function(event){
  console.log(event.detail.index, event.detail.fromTouch, event.detail.fromThumb);
}, false);

lightGallery(lg);
事件类型 参数 描述
onBeforeOpen.lg 立即启动开幕前被解雇.
onAfterOpen.lg 打开库后，立即解雇.
onAferAppendSlide.lg

event.detail.index -滑动指数

 当幻灯片内容已经被插入跌势容器此事件被触发.
onAfterAppendSubHtml.lg

event.detail.index -滑动指数

 （：标题/描述前）已追加到幻灯片当子HTML内容该事件被触发.
onSlideItemLoad.lg

event.detail.index -滑动指数

 一旦幻灯片里面的形象已经完全加载这个事件 .
onBeforeSlide.lg

event.detail.prevIndex -上一张幻灯片指数

event.detail.index - 滑动的索引

event.detail.fromTouch - 如果真的通过触摸事件呼吁鼠标拖动滑动功能

event.detail.fromThumb - 真如果通过点击缩略图叫滑动功能

 立即每张幻灯片转换之前解雇.
onAfterSlide.lg

event.detail.prevIndex - 上一张幻灯片指数

event.detail.index -滑动指数

event.detail.fromTouch - 如果真的通过触摸事件呼吁鼠标拖动滑动功能

fromThumb -真如果通过点击缩略图叫滑动功能

 每张幻灯片过渡后，立即解雇.
onBeforePrevSlide.lg

event.detail.index -滑动指数

event.detail.fromTouch -如果真的通过触摸事件呼吁鼠标拖动滑动功能

 立即每个“上一页”幻灯片切换之前触发
onBeforeNextSlide.lg

event.detail.index -滑动的索引

event.detail.fromTouch - 如果真的通过触摸事件呼吁鼠标拖动滑动功能

 立即每个“下一步”幻灯片切换之前触发
onDragstart.lg 当用户开始拖动滑块解雇
onDragmove.lg 拖动操作期间定期发射.
onDragend.lg 当用户已经完成了拖动操作发射
onSlideClick.lg 当用户点击幻灯片上，不越过刷卡的阈值被解雇.
onBeforeClose.lg 立即关闭过程开始前被解雇.
onCloseAfter.lg 一旦lightgallery被关闭马上还以颜色.

方法

如何使用方法 

var el = document.getElementById('lightgallery');

lightGallery(el);

// Go to third slide
// Index starts from 0
window.lgData[el.getAttribute('lg-uid')].slide(2);
名称 参数 描述
slide() index - 滑动的索引。从开始 0 转到特定的幻灯片。
goToNextSlide() 转到下一张幻灯片.
goToPrevSlide() 转到上一张幻灯片.
destroy() true -如果您传递true作为参数销毁方法将彻底摧毁画廊。如果参数未定义或假的破坏只会关闭库和插件实例保持与元素. 关闭或破坏图库

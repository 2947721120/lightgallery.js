带有自定义转换的图库.

lightgallery带有许多美丽的内置动画。您可以通过传递过渡名称来更改过渡类型的类型 mode 设置。你可以很容易地如果你的CSS知识创建自己的动画.

CSS的要求

lg-slide 和 lg-fade转换包括 lightgallery.css. 对于所有其他的转换，你需要包括 lg-animations.css 在你的文档中

  •

HTML 构造

<div id="custom-transitions">
  <a href="img/img1.jpg">
      <img src="img/thumb1.jpg" />
  </a>
  <a href="img/img2.jpg">
      <img src="img/thumb2.jpg" />
  </a>
  ...
</div>

Javascript

lightGallery(document.getElementById('custom-transitions'), {
    mode: 'lg-fade'
})

Youtube, Vimeo Video Gallery

Lightgallery支持的YouTube，Vimeo的，VK和所有其他类型的HTML5视频格式。如MP4，支持WebM，OGG，等.

要显示的YouTube，Vimeo的或VK视频，可以粘贴视频网址，或共享URL，这是由YouTube/ VIMEO的数据src属性提供。用同样的方法，你显示图库图片。Lightgallery将检查数据src属性，如果它是YouTube或VIMEO视频网址，它会创建视频幻灯片.

你甚至可以为每一个视频提供海报图像。海报图像将被加载，而不是视频。因此，用户将能够通过使用鼠标拖动或滑动导航到其他幻灯片。海报图像提高性能，并保持你的画廊的灵活性，而不会影响用户体验。当用户点击海报上的图像时，视频将被加载。可以将海报图像网址放置在数据海报属性中.

lightgallery允许你从YouTube自动加载缩略图，或是Vimeo VK。您可以指定在设置的缩略图的大小。当用户开始导航到另一个幻灯片时，视频将被自动暂停。所以用户永远不必担心。lightgallery照顾一切！

插件的依赖

Lightgallery要求视频插件包含在文档中.

API的依赖

Lightgallery需要VIMEO播放器API被包含在文档中的暂停VIMEO视频. https://f.vimeocdn.com/js/froogaloop2.min.js

视频播放器参数

这是很容易改变的视频播放器外观lightgallery。YouTube视频，你可以参考官方YouTube文档 here. For vimeo videos please refer the official vimeo documentation here.

Javascript

lightGallery(document.getElementById('video-player-param'), {
    youtubePlayerParams: {
        modestbranding: 1,
        showinfo: 0,
        rel: 0,
        controls: 0
    },
    vimeoPlayerParams: {
        byline : 0,
        portrait : 0,
        color : 'A90707'     
    }
});

Automatically load thumbnails

您可以自动加载YouTube的/ VIMEO视频的缩略图lightgallery缩略图插件的帮助。您可以轻松地通过传递缩略图大小指明缩略图的大小youtubeThumbSize/vimeoThumbSize 设置. Here is the full list of available thumbnails.

Plugin dependency

Lightgallery要求LG-缩略图插件可以在您的文档中显示缩略图.

  •

Javascript

lightGallery(document.getElementById('video-thumbnails'), {
    loadYoutubeThumbnail: true,
    youtubeThumbSize: 'default',
    loadVimeoThumbnail: true,
    vimeoThumbSize: 'thumbnail_medium',
});
